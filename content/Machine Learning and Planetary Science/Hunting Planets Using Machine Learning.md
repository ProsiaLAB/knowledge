### Motivation
Gas kinematics reveals still-forming planets in images of disks through features called *kinks*. 

### Basic Idea
[Terry+2022](https://iopscience.iop.org/article/10.3847/1538-4357/aca477) use [[Latin Hypercube|latin hypercube]] to distribute randomly-spaced data points evenly across the whole parameter space.
